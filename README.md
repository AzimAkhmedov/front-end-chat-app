
# AU Chat 

Front-end of Chat for AU students


## Run Locally

Clone the project

```bash
  git clone https://github.com/AKFA-IT/front-end-chat-app
```

Go to the project directory

```bash
  cd front-end-chat-app
```

Install dependencies

```bash
   yarn install
```

Start the server

```bash
   yarn dev
  
```
Open in browser
```bash
   go to http://localhost:3000/#
  
```
